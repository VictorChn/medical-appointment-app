medical-appointment-app

Objetivo general
Desarrollar una plataforma web integral para la gestión de citas médicas mediante el uso del framework Laravel, que permita automatizar la programación de consultas, optimizar la disponibilidad de los especialistas y mejorar la experiencia del paciente al reducir los tiempos de espera y facilitar el seguimiento clínico.

Tecnologias a utilizar
PHP
LARAVEL
COMPOSER
BLADE
MYSQL
TAILWIND
